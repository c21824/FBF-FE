## 🚀 How to Use

1. Open the app, login screen will be displayed
2. Enter email and password manually
3. Click **"Login"**
4. App will navigate to the corresponding role screen

## 🔒 Security

- Demo accounts are for development environment only
- Data is stored in memory, will be lost on app reload
- In production, real authentication with backend is required
